Installation

  1. Run these commands in an empty directory:
npm init
npx -p dyapi create
  1. Start the server:
npm run start
  1. It's done! ðŸŽ‰
  2. (If you want to update to the latest version, run npx -p dyapi update.)

Now, visit http://localhost:3000/api/users in your browser and you're supposed to see:

{
    "code": 200,
    "data": [],
    "total": 0,
    "pages": 0
}
